# 房地合一稅單機試算列印工具 V1

本工具為房地合一稅成交前初步試算工具，適合同事於案件討論、售屋前稅負評估、客戶溝通與內部教育訓練時使用。

## 使用網址

https://kai-5947.github.io/house-land-tax-calculator/

## 工具定位

本工具採用純前端 HTML / JavaScript 製作。

使用者輸入的資料只會在自己的瀏覽器畫面中計算，不會送出至資料庫，也不會儲存在 GitHub。

## 主要功能

1. 房地合一稅初步試算
2. 取得成本與出售費用扣除
3. 適用稅率選擇
4. 售屋後現金流試算
5. A4 列印與另存 PDF
6. 同事可直接透過網址使用
7. 可另存單機 HTML 離線備用

## 適用情境

- 售屋前初步稅負評估
- 屋主議價前現金流試算
- 仲介同事內部教育訓練
- 客戶初步說明
- 房地合一稅概算參考

## 目前版本

V1：基本試算版

目前包含：

- 案件基本資料
- 適用稅率選擇
- 出售價格
- 取得成本
- 取得可減除成本
- 出售可減除費用
- 課稅所得
- 預估房地合一稅
- 貸款塗銷金額
- 其他出售相關費用
- 預估售屋後可拿回現金

## 重要提醒

本工具僅供成交前初步評估參考。

實際稅額仍應以：

- 國稅局申報資料
- 國稅局核定結果
- 正式稅單
- 相關法令與個案事實

為準。

## 資料安全提醒

本專案不可上傳或保存任何真實客戶個資或敏感資料，包括但不限於：

- 客戶姓名完整資料
- 身分證字號
- 契約書
- 稅單
- 謄本
- 真實成交資料
- 貸款資料
- 私人財務資料

如需示範，請使用假資料或已去識別化的範例資料。
